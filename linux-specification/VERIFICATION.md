# Linux专题代码验证说明

## 概述

本文档说明如何验证Linux专题中的代码示例。

## 验证方式

### 方式一：使用验证脚本（推荐）

每个章节都有独立的验证脚本，位于linux-specification目录下：

```bash
# 进入专题目录
cd linux-specification

# 给脚本添加执行权限
chmod +x verify-*.sh

# 运行验证脚本
./verify-01.sh    # 验证第一章代码
./verify-02.sh    # 验证第二章代码
# ... 以此类推
```

### 方式二：手动验证

每个代码示例都可以单独复制到Linux终端执行。

## 注意事项

1. 部分命令需要root权限，使用`sudo`执行
2. 磁盘操作命令可能造成数据丢失，请谨慎使用
3. 建议在测试环境或虚拟机中运行验证
4. iptables/firewalld操作需注意防火墙规则

## 故障排除

如果验证失败：

1. 检查命令是否存在
2. 检查权限是否足够
3. 查看错误输出信息
4. 确认系统环境（Ubuntu/CentOS等）
