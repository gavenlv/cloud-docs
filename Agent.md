# Agent Prompt规则总结

## 概述

本文档定义专家级技术专题的生成规范。以GCP专题为标准，其他专题应达到同等深度和实操性。

---

## 核心规则

### 1. 语言一致性
- 使用中文（Simplified Chinese）进行所有回复
- 代码注释使用中文
- 保持语言一致性

### 2. 内容深度要求（最重要）
- **不只做表面信息展示，要深入底层原理**
- 解释技术基础而非仅仅"如何做"
- 每个概念都要讲清楚"为什么"
- 包含架构图、时序图、数据流图

### 3. 代码示例要求
- 每个章节的代码独立、可运行
- 必须包含验证步骤
- 代码要完整可用

### 1.2 虚拟化底层原理（KVM + EPT）
```
┌─────────────────────────────────────────────────────────────────┐
│  x86 Ring级别：Ring 0/1/2/3                                      │
│  硬件辅助虚拟化：Intel VT-x / AMD-V                              │
│  内存虚拟化：EPT（Extended Page Tables）                         │
│  CPU虚拟化：VMX root/non-root mode                               │

```
┌─────────────────────────────────────────────────────────────────┐
│  六种namespace：PID/Network/Mount/UTS/IPC/User                   │
│  cgroups控制器：cpu/memory/io/pids                               │
│  overlayfs三层：upperdir/lowerdir/merged                         │
│  copy-on-write机制                                               │
└─────────────────────────────────────────────────────────────────┘
```

### 1.4 HTTPS/TLS原理（非对称+对称混合）
```
┌─────────────────────────────────────────────────────────────────┐
│  对称加密：AES、DES、ChaCha20                                   │
│  非对称加密：RSA、ECDH                                           │

│  数字证书：CA签发、X.509格式                                     │
└─────────────────────────────────────────────────────────────────┘

**使用说明**

1. 使用触发语句：`同样的要求，做一份{主题名}`
2. 按照目录结构创建10个章节
3. 每个章节按照模板填充内容
4. 重点关注"为什么"而非"怎么做"
5. 添加完整的代码示例和验证步骤
6. 包含架构图和底层原理说明
